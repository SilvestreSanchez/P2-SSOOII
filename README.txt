Los archivos fuente se encuentran en el diretorio src
Los archivos de libros para probar la ejecucion se encuentran en el directorio utils
Los archivos ejecutables se encuentran en el directorio exec
Los archivos objeto se encuentran en el directorio obj


----- COMPILACION Y EJECUCION ------

Se ha utilizado Make para la automatizacion
En el raiz, ejecute "make" para compilar

Para ejecutar el programa:
"make solution nombre_archivo palabra n_hilos"
nombre_archivo debe ser la ruta absoluta del archivo

Para nombres de archivos largos, se recomienda ejecutar desde el directorio exec, ya que permite tabular y completar texto con TAB
(No siendo asi con make solution)
